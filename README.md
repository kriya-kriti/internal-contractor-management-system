
# Internal Contractor Management System

## Overview
This project is a front-end simulation of an internal contractor hiring and management system used within enterprise organizations. It represents internal workflows rather than a public-facing application.

The system models how managers, interviewers, and operations teams collaborate during requirement creation, interview evaluation, lifecycle tracking, and compliance review.

## Application Structure
- index.html: Central dashboard and navigation hub
- requirement_request.html: Requirement intake form for hiring managers
- interview_feedback.html: Structured interview evaluation form
- requirement_lifecycle.html: Requirement status and traceability view
- evaluation_compliance.html: Governance, evaluation, and compliance overview

## Key Highlights
- Enterprise-style dashboard layout
- Workflow-based UI design
- Structured data capture concepts
- Backend-ready page separation
- Compliance and audit awareness

## Technologies Used
- HTML5
- CSS3

## Intended Audience
This project demonstrates enterprise UI thinking and internal system design for front-end and full-stack developer roles.
